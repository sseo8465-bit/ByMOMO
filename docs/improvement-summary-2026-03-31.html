1. Branding (8.5 → 9.5)

  • 텍스트 전용 로고 → 발바닥 모티프 SVG 로고 리디자인 (MOMO의 두 번째 O에 발바닥 아이콘)
  • 로고 컴포넌트화: sm/md/lg 3단계 반응형 사이즈, currentColor 상속
  • GNB + Footer 일관된 로고 적용
  • Footer 네비게이션 라벨 통일 (About → Story)

2. UI/UX (6.0 → 8.5)

  • P0 상품 상세: +/− 수량 조절 UI 추가 (min 1, max 10, 실시간 합계)
  • P0 빈 장바구니: 듀얼 CTA ("맞춤 추천 받기" + "전체 제품 보기")
  • New 메인 페이지: "지금 가장 사랑받는 레시피" 3종 카드 섹션 (상황 라벨 + 해시태그)
  • New Best Picks 카드: 해시태그 피약 (#알러지프리, #피부건강 등)
  • New 이솝 스타일 원재료 함량 그래프 (Composition): Serif 퍼센트 숫자 + 수평 바 + 역할 표기
  • 상품 이미지 라운딩 (rounded-[6px]) — 메인, Shop, 추천 결과 페이지 전체 적용
  • 추천 결과 페이지 전면 개편: 한글 명조 타이틀, 프로필 정보 축소, 카드 간소화, Sticky Bar 재디자인
  • Hero CTA: "맞춤 간식 찾기" → "우리 아이 맞춤 추천 시작하기" (친근한 한글)

3. UX Copy (5.5 → 8.5)

핵심 원칙: "세련된 겉모습에 친절한 말투" — GNB는 영문(Story / Shop / Bag / My) 유지, 버튼·안내문은 다정한 한글
  • GNB: Cart → Bag (영문 유지, 패션 브랜드 감도)
  • 로그인 페이지: Login → "로그인", ID → "이메일", Password → "비밀번호"
  • 회원가입: "By MOMO 가족 되기" → "By MOMO 가족이 되기" (자연스러운 조사)
  • 장바구니 빈 상태: "아직 장바구니가 비어 있어요." + "우리 아이에게 맞는 간식을 골라볼까요?"
  • About 페이지: "연락처를 남겨주시면 순서대로 답변" → "아이의 이름과 문의 내용을 남겨주시면, 24시간 이내에 답변"
  • 뒤로 가기: "← Back" → "← 제품 목록으로"
  • 아이디/비밀번호 찾기: "Forgot ID" → "아이디 찾기", "Forgot Password" → "비밀번호 찾기"
  • 추천 결과 페이지: 카드 위 반복 문구 제거, 미니멀 해시태그로 대체

4. SEO (3.5 → 8.5)

가장 큰 폭으로 개선된 영역. 0점에 가깝던 메타 정보가 전 페이지에 체계적으로 적용됨.

  • Foundation layout.tsx: title template "%s | By MOMO", 공통 meta description, keywords 15개+
  • OG Open Graph: 전 페이지 og:title, og:description, og:image, og:type, og:locale(ko_KR)
  • OG 카카오톡/인스타 공유 시 "내 아이를 위한 단 하나뿐인 선물, By MOMO" 문구 + 브랜드 이미지 표시
  • Twitter Twitter Card: summary_large_image 전 페이지 적용
  • JSON-LD Organization 스키마 (layout.tsx)
  • JSON-LD FAQPage 스키마 (faq 페이지)
  • JSON-LD Product 스키마: 상품별 동적 generateMetadata + 고유 title/description/OG
  • 페이지별 고유 metadata: about, shop, faq, guide, terms, privacy, product/[id]
  • Canonical URL 설정 (alternates.canonical)
  • Footer Instagram 링크: nofollow 추가 (SEO 링크 주스 보존)

5. 마케팅 — 신뢰의 시각화

  • 상품 상세 "Composition" 섹션: 원재료 함량 비율을 이솝의 성분표처럼 정갈하게 시각화
  • 수평 바 그래프 (walnut → taupe → oatmeal 그라데이션) + Serif 대형 퍼센트 숫자
  • 각 원재료의 역할(Primary Protein, Omega-3, Joint Support 등) 영문 라벨 병기
  • "그래프조차 작품처럼 보여야 한다" → 달성

6. 맞춤 큐레이션 결과 페이지 개편

  • 메인 타이틀: font-serif-kr (한글 명조) + 사이즈 26/36/42px → 감성적 무게감
  • 반려견 프로필 정보: 10/11px 보조 텍스트로 축소
  • 큐레이션 설명: line-height 2.2, font-light, 최대 3줄 + 마지막 줄 이탤릭
  • 상품 카드: 이미지 위 반복 문구 삭제 → 미니멀 해시태그 피약 대체
  • 이미지 라운딩 rounded-[6px] 적용
  • 상품명 폰트: 16/18px font-semibold (명확한 위계)
  • Sticky Bar: charcoal(#2D221B) 배경, rounded-[8px], tracking-[0.14em], 충분한 여백
